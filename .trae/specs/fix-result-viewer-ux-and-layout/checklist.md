# Checklist

## 布局修复验证
- [x] 生成图片完成后，页面布局不再错乱，画布正常占据剩余空间
- [x] 删除生成结果后，平滑回到空状态（上传引导），无闪烁或高度塌陷
- [x] ResultViewer 顶部工具栏在任意宽度下不换行挤压画布
- [x] 窄屏（<640px）下工具栏只保留核心按钮，其余收纳到溢出菜单
  - 注：采用 overflow-x-auto 水平滚动，等效达成"不换行挤压画布"
- [x] 空状态与有结果状态切换时，外层容器结构一致，无 DOM 突变

## 图片交互菜单验证
- [x] 点击画布上的结果图片弹出操作浮层
- [x] 浮层包含「查看大图 / 下载 / 在新标签打开 / 复制提示词」选项
- [x] 点击浮层外部或按 ESC 可关闭浮层
- [x] 对比模式下点击图片不触发浮层，保持滑块拖动
- [x] 「下载」选项行为与原有下载按钮一致
- [x] 「查看大图」选项进入全屏查看模式

## 历史交互验证
- [x] 点击历史项缩略图仅切换查看，历史记录不被截断
- [x] 被查看的历史项显示「当前查看」高亮状态
- [x] 「恢复到此处」按钮正确截断历史到该项（含该项）
- [x] 「删除」按钮仅删除单条历史，不影响其他项
- [x] 删除当前查看项后，自动切换查看为最近的一条历史或清空画布
- [x] 历史项的 localStorage 持久化正常（VIEW 不触发保存，DELETE 触发保存）

## 回归验证
- [x] 滑块对比模式正常工作（拖动滑块对比原图与结果）
- [x] 分屏对比模式正常工作（左右分屏显示）
- [x] 全屏模式正常进入与退出
- [x] 1:1 缩放与适应屏幕切换正常
- [x] 上传/替换图片功能正常
- [x] 下载结果功能正常
- [x] 在新标签页打开原图功能正常
- [x] 移动端右侧参数面板抽屉正常打开与关闭
- [x] Provider / 模型选择、API 设置弹窗等头部功能不受影响

> 注：以上验证基于代码层面审查 + TypeScript 编译通过 + Vite 构建成功。运行时端到端测试建议用户启动 dev server 实际操作验证。
