## 修图总纲

**目标**：80%完成度的基准精修，自然精致、保留个人特色，不做过度标准化网红脸处理。

**修图哲学**：
- 修饰不是整容，保留"本人感"和自然真实感
- 比例调整要联动协调（缩小脸 → 鼻梁也对应缩小，保持均衡）
- 皮肤处理 = 去瑕疵不去纹理，缎面质感不是塑料感
- 保留个人特色的轻微不对称和独特瑕疵

---

## 一、脸型与轮廓

| 要求 | 程度 | 限制 |
|------|------|------|
| 优化脸型流畅度 | 轻微 | 不要明显改变原本脸型和骨相特征 |
| 收窄外轮廓 | 轻微 | 让面部线条更干净顺滑 |
| 缩小脸部视觉面积 | 适度 | 需同步协调五官比例，避免五官显得过大或位置失衡 |

**提示词语句**：
```
轻微收窄外轮廓，脸型线条流畅自然，面部干净顺滑。适度缩小脸部视觉面积，同步协调五官比例。不要改变原本脸型和骨相特征，不要过度瘦脸。
```

---

## 二、五官比例

| 要求 | 程度 | 限制 |
|------|------|------|
| 调整三庭五眼比例 | 微调 | 不大幅改变五官分布，使整体均衡自然 |
| 放大眼睛 | 稍微 | 保持原本眼型和神态 |
| 缩小鼻翼 | 适度 | 联动缩小鼻梁宽度以保持比例 |
| 抬高鼻梁 | 轻微（如偏低） | 不改变鼻子的个人特征 |
| 嘴巴/下巴/面中关系 | 保持自然协调 | 不改变布局 |

**提示词语句**：
```
微调三庭五眼比例至均衡自然，整体五官分布和谐，不大幅改变五官分布。稍微放大眼睛，保持原本眼型和神态。适度缩小鼻翼，鼻梁对应缩窄保持比例协调。如鼻梁偏低则轻微抬高，不改变鼻子的个人特征。嘴巴、下巴、面中关系保持自然协调。
```

## 三、皮肤处理

| 要求 | 程度 | 限制 |
|------|------|------|
| 去除痘印/斑点/泛红 | 干净去除 | 保留真实皮肤纹理和毛孔 |
| 改善暗沉和肤色不均 | 均匀肤色 | 不要磨皮过度 |
| 皮肤质感 | 高级缎面妆感 | 干净、通透、细腻、有自然光泽，不塑料不硅胶 |

**提示词语句**：
```
去除面部痘印、斑点、泛红、暗沉和肤色不均，保留真实皮肤纹理和毛孔，不要磨皮过度。皮肤呈高级缎面妆感：干净、通透、细腻、有自然光泽，不硅胶不塑料。
```

---

## 四、面部结构

| 要求 | 程度 | 限制 |
|------|------|------|
| 强化面部立体感 | 轻微 | 不僵硬不生硬 |
| 压暗外轮廓，提亮内轮廓 | 适当 | 结构更清晰但不过度 |
| 淡化法令纹/黑眼圈/疲惫感 | 淡化 | 保留自然表情，不让脸变僵 |

**提示词语句**：
```
轻微强化面部立体感，适当压暗外轮廓提亮内轮廓，面部结构更清晰但不生硬。淡化法令纹、黑眼圈和疲惫感，保留自然表情，不让脸变僵。
```

## 五、眉毛与细节

| 要求 | 程度 | 限制 |
|------|------|------|
| 清理眉毛杂毛 | 清理干净 | 不要重画眉毛 |
| 保留自然瑕疵和真实感 | 保留 | 不要追求完全无瑕 |

**提示词语句**：

```
清理眉毛周围杂毛，眉形干净自然，不重画眉毛。保留个人特色中的自然瑕疵和真实感，不追求完全无瑕。
```

## 操作流程

```
【Step 1】基准修图（80%）
  上传原图 → 上传皮肤参考图（如有）→ 使用完整提示词
  模型: GPT Image 2
  Denoising: 0.25 → 逐步提高到0.3 → 0.35

【Step 2】质检清单
  □ 脸型流畅自然？
  □ 鼻翼缩小且联动比例和谐？
  □ 痘印/斑点/法令纹是否去掉？
  □ 皮肤是否缎面质感且通透？
  □ 五官分布是否自然不僵硬？
  □ 个人特色是否保留？
  □ 整体是否像专业精修而非换脸？

【Step 3】微调（如需要）
  局部不满意 → 局部重绘 (inpaint)
  整体差一点 → 降低Denoising到0.2再跑一轮
```
