## 一、核心理论：通透的本质

### 1.1 画面不通透的两大根源

| 根源 | 表现 | 直方图特征 |
|------|------|-----------|
| **影调层次缺失** | 所有像素集中在中间调，没有纯黑和纯白，对比度低，像蒙了一层雾 | 像素堆积在中间，两端为空 |
| **色彩层次缺失** | 色彩饱和度低、色相单一，没有冷暖对比，色彩过渡生硬 | 色相分布集中，饱和度偏低 |

### 1.2 通透调色的黄金法则

> **先调光，后调色** — 先解决影调问题，再处理色彩问题，顺序绝对不能颠倒。

## 二、完整操作步骤拆解（可直接转化为提示词）

### 步骤 1：基础校正（必做第一步）

| 操作 | 作用 |
|------|------|
| 删除色差 | 修复色散，提升画面纯净度 |
| 启用配置文件校正 | 修复镜头畸变，从源头提升画质 |

### 步骤 2：影调去灰（核心中的核心）

| 操作 | 参数方向 | 作用 |
|------|---------|------|
| 提高对比度 | +15~+25 | 拉开明暗差距，打破中间调堆积 |
| 降低高光 | -30~-50 | 恢复高光细节，避免过曝 |
| 提高阴影 | +20~+40 | 恢复暗部细节，避免死黑 |
| 向左滑动黑色 | -10~-20 | 建立纯黑场，增强画面通透感 |
| 去除薄雾 | +5~+15 | 直接去除画面灰雾，提升清晰度 |

### 步骤 3：曲线精细调整

RGB曲线做轻微S型，进一步增强对比度，让画面更有层次感。

### 步骤 4：色彩层次塑造

**混色器调整**：
- 黄色色相左移 → 偏黄的绿色变成金黄色
- 绿色色相右移 → 绿色更翠绿
- 增加黄/绿色饱和度 → 色彩更浓郁

**颜色分级（二级调色关键）**：

| 区域 | 色相 | 饱和度 |
|------|------|--------|
| 阴影 | 150°左右（青绿色） | 低饱和度 |
| 中间调 | 30°左右（橙黄色） | 中饱和度 |
| 高光 | 40°左右（暖黄色） | 低饱和度 |

**校准面板**：蓝原色色相左移一点，提升整体色彩通透度。

### 步骤 5：蒙版光影塑造（画龙点睛）

1. 线性渐变压暗画面四角和边缘
2. 镜像滤镜提亮画面中心主体

作用：引导视觉焦点，增强画面立体感和层次感。

## 四、基于视频原理的 Image2 修图高级技巧

### 4.1 分步骤修图法（效果远超一次性修图）

#### 第一步：影调去灰

**正向提示词**：
```
去除画面灰雾，提高对比度，建立纯黑场和纯白场，保留所有高光和暗部细节
```

**负面提示词**：
```
画面发灰, 灰蒙蒙, 蒙雾感, 高光过曝, 暗部死黑
```

#### 第二步：色彩调整

**正向提示词**：
```
调整色彩层次，阴影偏青绿色，中间调和高光偏暖橙黄色，色彩过渡自然和谐
```

**负面提示词**：
```
色彩暗淡, 色彩饱和度低, 色彩层次单一, 色彩溢出
```

#### 第三步：细节增强

**正向提示词**：
```
锐化眼睛、眉毛、头发丝和衣服纹理，细节清晰锐利，锐化适度
```

**负面提示词**：
```
模糊, 噪点, 过度锐化
```

### 4.2 强度控制技巧

| 问题 | 解决方案 |
|------|---------|
| 画面还是有点灰 | 提示词加 `强烈去除薄雾` |
| 对比度太高 | 加 `对比度适中，光影过渡柔和` |
| 色彩太艳 | 加 `低饱和度，高级灰调` |

## 五、关键认知升级

1. **通透 ≠ 亮**：通透是有完整的影调层次和色彩层次，不是简单提亮
2. **AI修图最常见痛点**：绝大多数AI修图会把画面修得发灰、没有层次，本体系给出精准解决方案
3. **二级调色是"好看→高级"的关键**：颜色分级（阴影青绿 + 中间调/高光暖橙黄）是让AI修图从"好看"变成"高级"的核心
