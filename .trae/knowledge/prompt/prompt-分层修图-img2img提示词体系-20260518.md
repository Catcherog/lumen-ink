## 核心理念

**关键不是提示词越长越好，而是把"要告诉模型：保留什么 + 修改什么 + 参考哪种风格 + 不能改什么。**

---

## 一、最稳做法：原图 + 局部遮罩 + 风格参考图

### 推荐流程
1. 上传原图作为主图
2. 用遮罩圈出要修改的区域
3. 提示词只描述"被遮罩区域怎么改"
4. 明确要求保留人物、光线、构图、相机角度、画质风格
5. 加一张风格参考图，用来统一色调、质感、摄影风格

### 典型示例
```
保持原图中人物的脸部、五官、姿势、镜头角度、光线方向和背景透视不变。只修改遮罩区域：将衣服改成黑色高级缎面连衣裙，质感柔软，有自然褶皱，符合原图光影。整体风格参考第二张图：高级时尚杂志摄影、柔和电影感、低饱和、细腻皮肤质感。不要改变人物身份，不要改变脸型，不要改变发型，不要改变身体比例。
```

## 二、分层提示词写法（核心模板）

### 标准格式

```
【保留】
保持原图人物身份、脸部特征、姿势、构图、镜头角度、光线方向、背景透视不变。

【修改】
只修改：把白色T恤换成黑色皮夹克，合身，有真实皮革纹理和自然高光。

【风格】
整体风格参考图2：电影感人像摄影，柔和自然光，低对比，高级灰色调，真实皮肤纹理。

【限制】
不要改变脸、发型、眼睛、鼻子、嘴巴、身体比例、手部结构。不要卡通化，不要过度磨皮，不要生成额外人物。
```

### 为什么分层写？
- 模型更容易理解任务边界
- 不会把不需要改的部分也改了
- 风格和内容分离，更可控


## 四、参考图分类使用

| 参考图类型 | 用途 | 说明 |
|-----------|------|------|
| 人物原图 | 锁定身份、脸、姿势、构图 | 主图，用来保持人物一致性 |
| 风格参考图 | 色调、光影、质感、摄影风格 | 告诉模型"调色方向" |
| 服装参考图 | 衣服款式，不参考脸和身材 | 只参考款式和材质 |
| 背景参考图 | 环境氛围，不改变主图人物 | 只参考氛围和透视 |

### 示例

```
图1作为主图，保持人物身份和构图。图2只参考服装款式，不参考人物长相。图3只参考整体色调和电影感光影。
```

## 五、常见需求对应提示词模板

### 5.1 换衣服但保留人物

```
保持人物脸部、发型、姿势、身材比例、背景和光线不变。只将衣服替换为参考图中的黑色西装外套，剪裁合身，面料有真实纹理和自然褶皱，符合原图光影。不要改变人物身份，不要改变脸，不要改变手。
```

### 5.2 统一成杂志大片风格

```
保持原图人物身份、姿势和构图不变。将整体画面调整为高端时尚杂志摄影风格，柔和棚拍光线，细腻真实皮肤质感，低饱和高级色调，背景干净，画面清晰自然。不要过度磨皮，不要卡通化。
```

### 5.3 修脸但不像换脸

```
保留原人物身份和五官结构，只进行自然美化：轻微均匀肤色，减少皮肤瑕疵，保留真实皮肤纹理，眼神更清晰，面部光影自然。不要改变脸型、鼻子、嘴巴、眼睛形状，不要变成另一个人。
```

### 5.4 产品图保持品牌风格

```
保持产品形状、Logo、材质、比例和透视不变。优化灯光和背景，使画面更干净高级。整体风格参考图2：极简商业摄影，柔和反光，浅灰背景，真实材质质感。不要改变产品结构，不要重绘文字和Logo。
```

### 5.5 修衣服（结合参考图）

```
以图1为主图进行修图。保持人物身份、脸部特征、姿势、构图、镜头角度、光线方向和背景透视不变。

只修改衣服：将原来的白色上衣改成参考图2中的黑色高领针织衫，贴合身体，有真实针织纹理和自然褶皱。

参考图3只用于整体风格：低饱和电影感、柔和自然光、高级灰色调、真实皮肤质感。

最终效果真实自然，像同一次拍摄完成的照片。不要改变脸、发型、身体比例、手、背景、光线方向。
```

## 七、核心原则总结

```
原图锁身份和结构
遮罩锁修改范围（仅修遮罩区域）
参考图锁风格
提示词锁边界（说清楚不能改什么）
```
