## 一、核心提示词（主版本）

### 1.1 完整正向提示词

```
基于原图进行高强度但真实自然的人像精修和电影感调色。

保留人物身份、原始表情、眼神方向、发型、服装、姿态和背景构图，不要生成陌生脸，不要改变五官辨识度。

脸部进行自然精修：轻微小V脸，下颌线收紧，脸颊轮廓更流畅，中庭略微缩短，鼻翼和鼻头轻微缩小，人中略微缩短，面部紧致度提升。五官比例更精致，但保持本人特征，不要网红脸，不要过度整形感。

皮肤处理为高级韩系奶油肌：肤色提亮半到一档，肤色更均匀，去除明显暗沉、瑕疵和粗糙感，保留真实皮肤纹理、毛孔和面部立体光影。不要塑料皮，不要过度磨皮。

整体调色为新中式电影感写真：暗调、低饱和、高级灰绿色环境色，暖色窗光和肤色形成柔和对比，暗部更深但保留细节，画面有胶片质感、柔和高光、细腻颗粒感、东方含蓄氛围。整体氛围安静、克制、雅致，有中式庭院/花艺/茶室般的高级感。

输出高质感4K人像细节，面部清晰，背景保持自然虚化和原有空间关系。
```

### 1.2 完整负面提示词（必须每次都加）

```
不要换脸，不要改变人物身份，不要改变表情，不要改变发型，不要改变服装，不要改变背景结构，不要重画成陌生人，不要欧美化，不要二次元，不要网红蛇精脸，不要过度瘦脸，不要过度磨皮，不要塑料皮，不要假白，不要强烈美颜滤镜，不要柔焦糊脸，不要五官漂移，不要眼睛变大过度，不要嘴型变化过大，不要背景漂移，不要衣服变形，不要手部变形。
```

## 三、提示词结构方法论

### 3.1 六层结构模型

```
第1层：总体定位 → "高强度但真实自然的人像精修和电影感调色"
第2层：身份保护 → 7项锁定（身份/表情/眼神/发型/服装/姿态/背景）
第3层：脸部精修 → 逐项量化（脸型/下颌/中庭/鼻翼/人中/紧致度）
第4层：皮肤处理 → 质感优先（韩系奶油肌 + 保留纹理毛孔）
第5层：调色风格 → 差异化标签（新中式暗调电影感）
第6层：输出质量 → 交付标准（4K + 背景虚化）
```

### 3.2 攻守兼备策略

```
【攻】正向提示词 = 告诉AI要做什么
    ├─ 做什么（精修+调色）
    ├─ 怎么做（逐项量化）
    └─ 做到什么程度（轻微/半档/自然）

【守】负面提示词 = 告诉AI绝对不能做什么
    ├─ 不要换脸（身份防线）
    ├─ 不要过度（质感防线）
    ├─ 不要改变背景（构图防线）
    └─ 不要网红化（审美防线）
```

### 3.3 反复强调策略

关键约束在正向和负面提示词中**双重锁定**：
- 正向："保留人物身份" + "保持本人特征"
- 负面："不要换脸" + "不要改变人物身份" + "不要重画成陌生人"

## 四、四大翻车点与防御策略

| 翻车点 | 表现 | 防御策略 |
|--------|------|---------|
| **脸修成网红** | 蛇精脸、过度整形 | 提示词强调"轻微"、"保持本人特征"、"不要网红蛇精脸" |
| **肤色太白** | 假白、塑料皮 | 提示词强调"保留真实纹理"、"不要假白"、"不要塑料皮" |
| **背景被重绘** | 背景漂移、空间关系改变 | 提示词强调"保留背景构图"、"不要改变背景结构" |
| **光影变滤镜** | 小红书滤镜感、柔焦糊脸 | 提示词强调"保留面部立体光影"、"不要柔焦糊脸" |
