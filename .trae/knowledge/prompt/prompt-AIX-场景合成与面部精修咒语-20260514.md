## 三、✨ 面部精修咒语（2号节点使用）

### 3.1 自然美颜咒语（**通用版**）

**核心优势**：比主流AI修图软件更便宜，效果更自然

```
把脸P好看，窄脸、缩颧骨、缩短中庭，缩短人中，缩小鼻翼，平整光滑的皮肤，流畅的脸型，不要改变妆容，不要改变其他任何元素
```

**可微调参数**：

| 调整项 | 修改示例 | 效果 |
|-------|---------|------|
| 脸型 | `窄脸` → `鹅蛋脸` / `瓜子脸` | 改变脸型轮廓 |
| 中庭 | `缩短中庭` → 保持或删除 | 调整面部比例 |
| 人中 | `缩短人中` → 保持或删除 | 调整唇上部比例 |
| 鼻翼 | `缩小鼻翼` → `挺拔鼻梁` | 调整鼻子形态 |
| 皮肤 | `平整光滑的皮肤` → `奶油肌` / `哑光肌` | 调整肤质表现 |

**重要约束**：
- ✅ `不要改变妆容` - 保留原有化妆效果
- ✅ `不要改变其他任何元素` - 只调整面部，不影响背景、服装、发型等

## 四、咒语结构分析（供Image2/Banana适配参考）

### 4.1 场景合成咒语的标准化结构

```
[画质标准] + [风格定义] + [光影描述] + [场景元素] + [构图要求] + [色彩控制] + [物理约束] + [身份保护]
```

**拆解示例（花瓣纷飞场景）**：

| 模块 | 内容 | 作用 |
|------|------|------|
| 画质标准 | 电影级写实摄影质感，特写镜头高清 | 定义输出质量 |
| 风格定义 | 游戏CG风格，极具艺术感，震撼人心 | 设定艺术方向 |
| 光影描述 | 真实世界物理光影，暖色丁达尔发丝光，光影交错 | 控制光线逻辑 |
| 场景元素 | 白色花瓣纷飞，前景白色花瓣虚化，背景动态虚化 | 添加合成元素 |
| 构图要求 | 背景压暗，暗部叠加，特写镜头低饱和 | 控制画面构图 |
| 色彩控制 | 低饱和高级纪实色调 | 统一色调 |
| 物理约束 | 光线逻辑符合真实世界，面部亮度合理 | 保证真实感 |
| 身份保护 | 保证人物表情不变，姿势不变，构图角度不变，原比例 | 保护主体特征 |

### 4.2 面部精修咒语的标准化结构

```
[指令动词] + [具体调整项] + [质量标准] + [约束条件]
```

**拆解示例（自然美颜咒语）**：

| 模块 | 内容 | 作用 |
|------|------|------|
| 指令动词 | 把脸P好看 | 明确操作意图 |
| 具体调整项 | 窄脸、缩颧骨、缩短中庭... | 列出所有修改点 |
| 质量标准 | 平整光滑的皮肤，流畅的脸型 | 定义最终效果 |
| 约束条件 | 不要改变妆容，不要改变其他任何元素 | 限制修改范围 |

### 5.2 氛围调整技巧

**明亮欢快**：
- 关键词：阳光灿烂、明亮、温暖、活力
- 色调：暖色调、高饱和
- 光线：丁达尔光、顺光、填充光

**神秘忧郁**：
- 关键词：雾气、阴郁、朦胧、情绪化
- 色调：冷色调、低饱和、暗调
- 光线：侧逆光、轮廓光、低调照明

**知性文艺**：
- 关键词：图书馆、书香、安静、思考
- 色调：中性灰、低对比度
- 光线：柔和窗光、散射光、自然光

### 5.3 ⚠️ 绝对不能删除的部分

无论怎么自定义，以下内容**必须保留**在咒语末尾：

```plaintext
保证人物表情不变
姿势不变
构图角度不变
原比例不变
（或类似表述）
```

**原因**：这些约束条件是保证AI不会过度修改人物特征的关键，删除后可能导致：
- 面部特征失真
- 姿势变形
- 构图崩坏
- 比例失调

## 六、Sotlab平台适配建议（待验证）

### 6.1 Image2适配要点
- ✅ Image2的文字理解能力强，可以直接使用完整咒语
- ✅ 建议开启"思考模式"提升理解精度
- ✅ 可利用Image2的"局部编辑"功能进行精细化调整
- ⚠️ 可能需要调整咒语中的某些参数表述以匹配Image2的最佳实践

### 6.2 Banana适配要点
- ✅ Banana对中文提示词支持良好
- ⚠️ Banana的文字理解能力较弱，建议简化咒语结构
- ⚠️ 可能需要将长咒语拆分为多个短提示词
- ✅ Banana在皮肤质感方面表现优秀，适合面部精修节点

### 6.3 参数优化方向
- **CFG Scale**：建议7-9（平衡创意性和遵循度）
- **Steps**：建议30-50步（保证质量）
- **Denoising Strength**：0.4-0.6（场景合成）、0.3-0.5（面部精修）
- **Seed**：固定种子以保证系列作品一致性
