### 一、思考模式 (Thinking) 使用技巧

#### 核心能力：
- 开启后，Image2 会**先分析需求，再自动生成最优方案**
- 可以**联网搜索实时信息**
- 一次性生成**最多 8 张**保持角色和风格完全一致的系列图片

#### 适用场景：
- 复杂的多步骤任务
- 需要系列化输出的场景
- 需要保持角色和风格一致性的批量生成

#### 实战案例：
```
需求："帮我设计一套新中式写真的宣传海报，包括主海报、朋友圈九宫格和价格表"
→ 思考模式自动规划 → 一次性输出完整系列
```

#### 写真业务最佳实践：
- **写真套系生成**：开启思考模式，一次性生成同一客户的 8 张不同姿势和场景的写真
- **宣传物料批量**：主海报 + 朋友圈九宫格 + 价格表，一套输出

### 二、文字精准渲染技巧

#### 核心能力：
- 中文文字准确率达到 **99% 以上**
- 支持多种字体：**宋体、黑体、楷体、毛笔字**等
- 可指定文字的**大小、颜色、位置和排版方式**
- 遥遥领先于 Banana 和其他所有模型

#### 渲染提示词模板：
```
"文字清晰无畸变，排版规范，符合设计美学"
```

#### 写真业务应用：
- 写真海报（带客户姓名、日期、主题文字）
- 客户相册封面
- 价格表与菜单
- 宣传单与活动海报
- 公众号封面图

### 三、角色锁定与一致性技巧

#### 核心方法：

**1. 定妆照生成法：**
- 同时生成**正面、侧面、背面**三张图
- 使用提示词：
  ```
  "保持人物面部特征、发型、服装完全一致，不同角度和姿势"
  ```

**2. 思考模式批量法：**
- 开启思考模式后，一次性生成同一角色的多个不同动作和场景
- 自动保持角色一致性

**3. 角色资产库锁定法：**
- 生成定妆照 → 添加到角色资产库 → 锁定角色
- 后续所有生成自动沿用锁定形象

#### 写真业务应用：
- 客户多角度写真（正面/侧面/背面）
- 同一客户的多场景套系
- 动态视频的关键帧准备
