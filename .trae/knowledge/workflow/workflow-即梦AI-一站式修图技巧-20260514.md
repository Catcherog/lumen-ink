## 三、三大核心修图技巧（分步详解）

### 3.1 技巧一：消除多余人物 + 细节修复

**解决痛点**：生成的图片中出现不需要的多余人物、杂物，或者面部、文字有错误

#### 详细操作步骤：

**Step 1：打开图片进入编辑模式**
- 导入需要修复的图片
- 点击编辑按钮进入编辑界面

**Step 2：使用消除画笔**
- 点击左侧工具栏的"**消除画笔**"功能
- 用画笔涂抹需要消除的部分

**✅ 操作要点**：
- 必须**完全覆盖**想要消除的区域
- 多涂出去一点没关系，系统会自动计算周围环境进行填充
- 宁可多涂不要少涂

**❌ 注意事项**：
- 不要只涂一半，否则会留下残留痕迹
- 涂抹边缘要清晰明确

**Step 3：执行消除**
- 点击消除按钮
- 等待3-5秒即可看到结果

**Step 4：细节修复（关键步骤！）**
- 检查消除后的画面
- 如果发现细节有问题（面部扭曲、文字乱码、招牌模糊等）
- 点击"**细节修复**"按钮
- 系统会自动扫描并修复整个画面的错误

**Step 5：对比保存**
- 修复完成后对比前后效果
- 满意即可保存

#### 实际效果验证：
✅ 完美消除了画面中多余的路人  
✅ 成功修复了牌坊上的乱码汉字，变成清晰可识别的文字  
✅ 修复了人物面部的崩坏问题，让表情和五官更自然  

---

### 3.2 技巧二：重新构图（改比例 + 移动主体 + 添加元素）

**解决痛点**：生成的图片比例不合适、主体位置不好，或者想要添加新元素丰富画面

#### 详细操作步骤：

##### **第一步：改变图片比例（如 1:1 → 16:9）**

1. 在编辑模式中，点击"**画布设置**"
2. 将比例改为需要的尺寸（如 16:9）
3. 此时图片会居中显示，**四周出现空白区域**
4. 点击左侧工具栏的"**扩图**"功能
5. （可选）在扩图描述框中输入想要的内容，不输入则系统自动根据现有画面延展
6. 点击扩图按钮，系统会生成 **4个不同的延展结果**
7. 选择最自然、最符合预期的一个，点击完成编辑

##### **第二步：移动主体位置**

**目标**：将主体从原位置移动到更理想的构图位置

1. 回到原图，使用**消除画笔**将主体人物**完全涂抹消除**
2. 点击消除，得到一个**完整干净的背景图**
3. 复制一份原图，用于**抠取主体**
4. 在复制的图上点击"**抠图**"按钮，系统会自动识别并选中主体
5. 点击确认抠图，主体会变成一个**独立的图层**
6. 将抠好的主体图层**拖到刚才做好的背景图上**
7. 自由拖动主体到想要的位置
8. 还可以进行以下操作：
   - **水平翻转**
   - **缩放**（放大/缩小）
   - **旋转**（调整角度）

##### **第三步：添加新元素**

1. 从外部导入想要添加的元素图片（如装饰物、道具、其他人物等）
2. 对导入的图片执行同样的"**一键抠图**"操作
3. 将抠好的新元素**拖到画布上**
4. 调整：
   - **大小**（缩放）
   - **位置**（拖动）
   - **角度**（旋转）
5. 调整各元素的**位置关系**，形成想要的构图

#### 核心优势：
✅ 不用为了调整构图重新生成整张图  
✅ 可以精确控制每个元素的位置和大小  
✅ 大大提高了构图的灵活性和可控性  
✅ 节省大量时间和成本  

---

### 3.3 技巧三：多图融合 + 风格统一

**解决痛点**：从不同图片中拼接元素时，出现风格不统一、边缘僵硬、光影不一致的问题

#### 方案A：抠图 + 熔图（简单快速版）

**适用场景**：要求不高的简单融合、边缘过渡问题

**操作步骤**：

1. **准备两张图**：
   - 图A：保留主体（需要提取的元素）
   - 图B：保留背景（目标底图）

2. **对主体图执行一键抠图**
   - 得到独立的主体图层

3. **将主体图层拖到背景图上**
   - 调整到合适位置

4. **检查融合效果**
   - 如果发现主体边缘僵硬、和背景融合不自然
   - 进入下一步

5. **使用熔图功能**
   - 点击左侧工具栏的"**熔图**"功能
   - 调整熔图参数：
     - **强度**：控制融合的程度
     - **边缘羽化**：控制边缘的柔和度
   - 点击熔图按钮
   - 系统会自动融合两个图层，让边缘过渡自然

**优点**：速度快，操作简单  
**缺点**：对于复杂的光影差异效果有限  

---

#### 方案B：图生图 + 图片信息参考模式（高级专业版）⭐推荐

**适用场景**：熔图后风格仍然不统一，或对整体画面的和谐度有很高要求

**操作步骤**：

**Step 1：准备拼接草图**
- 按照方案A的步骤
- 将所有元素摆放到合适的位置
- 得到一张**拼接好的草图**（可能风格不统一，但位置对了）

**Step 2：使用图生图功能**
- 点击左侧工具栏的"**图生图**"功能
- 上传刚才做好的拼接草图作为参考

**Step 3：选择图片信息参考模式**
- 在参数设置中，选择"**图片信息参考模式**"

**Step 4：调整关键参数 - 图片信息参考强度** ⭐核心！

这是整个技巧的**灵魂参数**，决定了最终效果的平衡点：

| 参考强度值 | 效果描述 | 适用场景 |
|-----------|---------|---------|
| **0.8 - 0.9** | 生成的图**非常接近**参考图（草图） | 想要完全保留元素位置和构图 |
| **0.5 - 0.7** | 平衡**保留元素**和**风格统一** | 最常用的平衡点，兼顾两者 |
| **0.2 - 0.4** | 生成的图**变化较大**，风格更统一但会丢失部分细节 | 想要大改风格但保留大致构图 |

**Step 5：（可选）添加风格提示词**
- 在提示词框中输入简单的风格描述
- 示例：
  ```
  电影质感，光影自然，色调和谐，
  画面统一，边缘过渡自然，高质量渲染
  ```

**Step 6：生成并筛选**
- 点击生成按钮
- 系统会生成 **4个不同的结果**
- 选择：
  - 风格最统一的
  - 元素保留最完整的
  - 整体效果最好的

**Step 7：最终优化**
- 使用"**细节修复**"功能优化小问题
- 得到最终成品

#### 进阶用法：

**用法1：多版本对比**
- 调整不同的参考强度值（如0.6、0.7、0.8）
- 生成多个版本
- 挑选最好的一个
- 或者提取不同版本的优点进行二次融合

**用法2：复杂场景制作**
- 特别适合**多人物、复杂场景**的图片
- 解决AI生图多人物容易崩坏的问题
- 可以分别生成单个元素，再通过此方法融合

**用法3：迭代优化**
- 第一轮：强度0.8（保位置）
- 第二轮：对不满意的部分，强度0.6（微调）
- 第三轮：强度0.4（强统一样式）

## 四、避坑指南与经验总结

### 4.1 核心原则

#### ✅ **原则1：不要盲目抽卡**
> 当生成的图只有**小问题时**，优先用**修复功能**，而不是重新生成。
> 
> 重新生成可能会出现**新的、更严重的问题**。

**正确做法**：
- 小瑕疵 → 消除画笔 + 细节修复
- 构图问题 → 扩图 + 抠图重排
- 风格不统一 → 熔图 / 图生图
- 大问题才考虑重新生成

#### ✅ **原则2：消除画笔宁多勿少**
> 涂抹的时候稍微**多涂一点**，系统填充的效果会更自然。
> 
> 如果**涂少了**，容易留下明显的痕迹。

**实操建议**：
- 涂抹范围要比实际需要消除的区域**大10-20%**
- 边缘部分要**完全覆盖**
- 不要担心涂多了，AI会智能处理

#### ✅ **原则3：细节修复是万能药**
> 几乎所有的**小问题**都可以用细节修复解决，而且效果非常好。

**适用问题列表**：
- ✅ 面部五官扭曲/崩坏
- ✅ 文字乱码/模糊
- ✅ 肢体结构错误
- ✅ 纹理异常
- ✅ 边缘锯齿
- ✅ 色块噪点

### 4.2 熔图 vs 图生图 选择指南

| 对比维度 | 熔图 | 图生图+图片信息参考 |
|---------|------|------------------|
| **适用场景** | 简单融合、边缘问题 | 风格不统一、光影不一致 |
| **操作难度** | ⭐⭐ 简单 | ⭐⭐⭐ 中等 |
| **处理速度** | 快（几秒） | 中等（十几秒到一分钟） |
| **效果质量** | 一般 | **优秀** ⭐ |
| **可控性** | 参数较少 | 参数丰富（参考强度） |
| **推荐指数** | ⭐⭐⭐ 快速场景 | ⭐⭐⭐⭐⭐ 高质量需求 |

**选择决策树**：
```
需要融合多个元素？
├── 只是边缘僵硬？ → 用熔图（快）
├── 光影/色调/风格不一致？ → 用图生图（效果好）
├── 要求快速出图？ → 先试熔图，不行再用图生图
└── 要求高质量成品？ → 直接用图生图+参考模式
```

### 4.3 图片信息参考强度设置速查表

| 目标 | 推荐强度 | 预期效果 | 风险提示 |
|------|---------|---------|---------|
| **完全保留元素位置和细节** | 0.85-0.95 | 几乎不变，只微调风格 | 可能风格统一不够 |
| **主要保留，轻微优化** | 0.70-0.80 | 保留大部分，小幅改善 | 平衡性好 |
| **平衡保留与风格** | 0.50-0.70 | 明显改善风格，保留主要元素 | 可能丢失小细节 |
| **侧重风格统一** | 0.30-0.50 | 风格高度统一，构图保留 | 元素可能有变形 |
| **大幅重绘** | 0.10-0.30 | 接近重新生成，仅保留轮廓 | 元素可能大变 |

**新手建议**：从 **0.7** 开始测试，根据效果上下调整

## 五、完整修图工作流总结

### 标准SOP（标准作业程序）

```
导入生图（AI生成的初始图片）
    ↓
【第一关：清理】
    ├─ 消除多余元素（消除画笔）
    └─ 细节修复（自动修复小问题）
    ↓
【第二关：构图】（可选，如果构图需要调整）
    ├─ 扩图改比例（如1:1→16:9）
    ├─ 抠图移动主体（重新安排主体位置）
    └─ 添加新元素（导入外部素材）
    ↓
【第三关：融合】（可选，如果有多元素拼接）
    ├─ 简单场景：熔图（快速融合）
    └─ 复杂场景：图生图+图片信息参考（高质量统一）
    ↓
【第四关：收尾】
    └─ 最终细节修复（优化残留小问题）
    ↓
导出成品 ✨
```

### 工作流变体

**变体1：快速修复版**（适用于小问题）
```
导入 → 消除画笔 → 细节修复 → 导出
耗时：1-3分钟
```

**变体2：重构构图版**（适用于构图问题）
```
导入 → 消除 → 扩图 → 抽主体 → 重排位置 → 熔图 → 细节修复 → 导出
耗时：5-10分钟
```

**变体3：精品制作版**（适用于高质量需求）
```
导入 → 消除 → 细节修复 → 扩图 → 抠图 → 多元素组合 → 
图生图(参考强度0.7) → 选最佳结果 → 再次细节修复 → 导出
耗时：10-15分钟
```

## 七、典型应用场景

### 7.1 AI写真正期修复

## 八、进阶技巧与参数优化

### 8.1 消除画笔的高级用法

**技巧1：分层多次消除**
- 对于复杂背景中的元素
- 分多次逐步消除
- 每次消除后检查效果
- 比一次性大面积涂抹更精准

**技巧2：保护重要区域**
- 消除前先用遮罩保护不想动的区域
- 避免误伤重要元素
- 特别是精细的边缘部分

**技巧3：结合手动修复**
- 消除后如果还有微小瑕疵
- 用克隆图章或修复画笔微调
- 达到完美无痕的效果

### 8.2 抠图精度提升

**技巧1：预处理提升识别率**
- 抠图前适当调整对比度和亮度
- 让主体和背景的边界更清晰
- 提高AI识别准确度

**技巧2：边缘精细化**
- 自动抠图后检查边缘
- 对于头发丝等细节
- 手动调整边缘蒙版

**技巧3：多层备份**

    如果元素丢失太多 → 强度提高到 0.8-0.85

## 九、常见问题FAQ

**Q1：消除后留下了明显的痕迹怎么办？**
A：尝试以下方法：
   - 用消除画笔再涂一次（扩大范围）
   - 使用细节修复功能
   - 如果还不行，用图生图局部重绘该区域

**Q2：抠图边缘不准确，怎么处理？**
A：
   - 检查原图对比度是否足够
   - 手动调整边缘蒙版
   - 使用熔图功能柔化边缘
   - 或用图生图重新生成该区域

**Q3：扩图后新增区域不自然怎么办？**
A：
   - 在扩图时输入具体的描述词引导
   - 使用图生图对新增区域进行优化
   - 或者裁掉不自然的部分重新扩图

**Q4：图生图后元素位置变了怎么办？**
A：
   - 提高图片信息参考强度至0.85以上
   - 或在提示词中强调"保持原有构图和元素位置"
   - 分区域逐个处理而非整体处理

**Q5：多个元素融合后看起来很假？**
A：
   - 检查每个元素的光源方向是否一致
   - 使用图生图+图片信息参考（强度0.5-0.7）进行全局统一
   - 添加统一的环境光和阴影效果
   - 最后用细节修复平滑接缝
