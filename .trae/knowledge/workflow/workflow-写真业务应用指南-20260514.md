### 一、写真业务六大应用场景

#### 1. 客户照片精修
- **使用功能**：局部修改
- **操作**：调整客户的肤色、身材、服装，去除瑕疵
- **模型选择**：GPT-Image2（局部编辑能力最强）

#### 2. 风格化处理
- **使用功能**：整体风格重绘 + 跨图风格迁移
- **操作**：将客户的写实照片转换成新中式、港风、复古等各种艺术风格
- **模型选择**：GPT-Image2

#### 3. 写真套系生成
- **使用功能**：思考模式 (Thinking)
- **操作**：开启思考模式，一次性生成同一客户的 8 张不同姿势和场景的写真
- **模型选择**：GPT-Image2（思考模式）

#### 4. 动态写真制作
- **使用功能**：Image2 关键帧 + Seedance2.0 视频生成
- **操作**：用 Image2 生成关键帧，Seedance2.0 生成 15 秒动态视频
- **定位**：客户的增值服务
- **模型选择**：GPT-Image2 → Seedance2.0

#### 5. 宣传物料制作
- **使用功能**：海报设计 + 文字渲染
- **操作**：生成工作室的海报、价目表、样册、社交媒体配图
- **模型选择**：GPT-Image2（文字渲染能力最强）

#### 6. 老照片修复
- **使用功能**：图片高清修复
- **操作**：将模糊、低分辨率的老照片修复成高清大图
- **模型选择**：GPT-Image2

### 二、AI 行为规范（应用要求）

> ⚠️ **以下规则为最高优先级，必须在每次写真业务相关对话中自动执行**

#### 规则 1：图片修改优先推荐局部编辑
- **触发条件**：用户需要修改图片
- **行为**：优先推荐使用 Image2 的局部编辑功能，而不是重新生成
- **原因**：局部编辑更精准、更高效、更保留原始特征

#### 规则 2：系列图片自动开启思考模式
- **触发条件**：用户需要生成系列图片（2张及以上）
- **行为**：自动推荐开启思考模式，确保角色和风格的一致性
- **原因**：思考模式可一次性生成最多 8 张完全一致的系列图

#### 规则 3：带文字设计明确说明 Image2 优势
- **触发条件**：用户需要生成带文字的设计
- **行为**：明确说明 Image2 的文字渲染能力是目前最强的（99%+ 准确率）
- **原因**：这是 Image2 相比 Banana 等模型的独家优势

#### 规则 4：长视频推荐 Image2+Seedance2.0 工作流
- **触发条件**：用户需要制作长视频
- **行为**：推荐使用"Image2 关键帧 + Seedance2.0 分段生成"的工作流
- **关键参数**：一致性强度设为"最高"，动作幅度设为"低"

#### 规则 5：所有新技巧自动应用到写真业务
- **触发条件**：任何时候
- **行为**：所有新学到的技巧都要自动应用到写真业务中，为用户提供更专业的解决方案
- **原因**：持续优化服务质量

### 三、写真业务服务流程（标准 SOP）

```
客户咨询
    ↓
需求分析（写真类型、风格偏好、数量、预算）
    ↓
【静态写真路线】
    上传客户照片 → 参考生图模式 → 生成多场景写真 → 精修优化 → 交付
    ↓
【动态写真路线】（增值服务）
    建立角色资产库 → 生成关键帧 → Seedance2.0 生成视频 → 拼接导出 → 交付
    ↓
【宣传物料路线】
    海报设计 → 价格表 → 社交媒体配图 → 样册 → 交付
```

### 四、模型选择决策树

```
需要做什么？
├── 修改已有图片的局部 → GPT-Image2（局部编辑）
├── 生成新的静态写真 → GPT-Image2（参考生图模式）
├── 批量生成系列写真 → GPT-Image2（思考模式）

├── 制作动态视频 → GPT-Image2（关键帧）→ Seedance2.0（视频）
├── 快速简单修图 → NanoBanana（备选）

```
