## 二、3节点自动化修图工作流（Shotlab版）

### 2.1 工作流架构

```
【输入】原图照片
    ↓
【节点A】场景合成 → 模型：GPT Image 2
    ↓ 原因：文字理解⭐⭐⭐⭐⭐，完美解析复杂场景咒语
    ↓ 输出：带光影场景的基础成片
【节点B】面部精修 → 模型：NanoBananaPro 或 GPT Image 2
    ↓ 原因：Banana皮肤质感⭐⭐⭐⭐⭐ / Image2局部编辑精确
    ↓ 输出：面部优化的成片
【节点C】清晰度提升 → 模型：GPT Image 2（高清修复）
    ↓ 原因：降低AI感，接近相机原生效果
    ↓
【输出】最终高清成片 ✨
```

### 2.2 推荐理由（基于模型特性）

**为什么节点A用Image2？**
- 场景合成咒语很长且复杂（50-80字）
- 包含大量专业术语："电影级写实"、"丁达尔发丝光"、"低饱和高级纪实色调"
- Image2的文字理解能力业界最强（⭐⭐⭐⭐⭐）
- 能准确理解每个修饰语的含义和权重

**为什么节点B用BananaPro？**
- 面部精修的核心需求是**皮肤质感**
- NanoBananaPro的皮肤表现业界最佳（⭐⭐⭐⭐⭐）
- 能产生真实的毛孔、纹理、光泽
- 避免"塑料脸"或"磨皮过度"

**为什么节点C用Image2？**
- 超分辨率需要智能填充细节
- Image2的高清修复算法成熟
- 能最大程度降低AI生成痕迹
- 让最终输出接近相机原生拍摄

## 三、Step-by-Step 操作指南

### Step 0：进入画布工作流

**操作路径**：
```
打开Shotlab网站 → 登录账号 → 点击底部"**画布**"标签 → 点击"新建画布"
```

**界面位置**：
- 从截图底部标签栏找到：`发现 | 图片 | 视频 | **画布** | 短片 | 正版素材`
- 点击"**画布**"进入节点式工作流界面

---

### Step 1：导入原图

**在画布中操作**：

1. **添加图片输入节点**
   - 在空白画布区域点击"+"号
   - 选择"上传图片"或"图片输入"
   - 从本地上传需要修图的客户照片

2. **放置与命名**
   - 将原图节点放在画布**左侧区域**
   - （可选）右键节点 → 重命名为"`01_原图输入`"
   - 作为整个工作流的起始源

---

### Step 2：创建节点A - 场景合成

#### 2.1 添加文本节点
- 在原图节点右侧点击"+"
- 选择"文本输入"或"提示词"
- 创建新的文本节点

#### 2.2 输入场景合成咒语

**⭐ 推荐：花瓣纷飞场景（成功率≥80%）**

直接复制以下完整提示词：

```plaintext
电影级写实摄影质感，真实世界物理光影，低饱和高级纪实色调，
游戏CG风格，极具艺术感，震撼人心，背景压暗，暗部叠加，
特写镜头高清，白色花瓣纷飞，前景少量白色花瓣虚化，
背景动态虚化，阳光灿烂，暖色丁达尔发丝光，光影交错，
面部亮度合理，特写镜头低饱和，光线逻辑符合真实世界，
保证人物表情不变，姿势不变，构图角度不变，原比例。
```

**其他可选场景**（详见prompt知识库）：
- ❄️ 户外雪地场景
- 📚 图书馆光影场景
- 🌫️ 雾气阴郁场景

#### 2.3 连接节点
- 从"01_原图输入"节点拉出连接线
- 连接到刚创建的文本节点
- 在弹出选项中选择"**参考生图**"或"**图生图**"

#### 2.4 设置参数

| 参数 | 推荐设置 | 说明 |
|------|---------|------|
| **模型** | `GPT Image 2` | 在模型下拉菜单中选择 |
| **分辨率** | `1K 标清` 或 `2K 高清` | 根据截图显示的尺寸菜单选择 |
| **CFG Scale** | 7.5-8.5 | 如果可调的话（平衡创意与遵循度）|
| **Steps** | 40-50 | 保证质量 |
| **Denoising Strength** | 0.45-0.55 | 中等强度，保留原图特征 |

#### 2.5 运行与筛选
- 点击节点上的"运行"或"生成"按钮
- 等待30秒-2分钟
- 查看生成的结果（可能有多张）
- 选择最满意的1张进入下一步

**💡 提示**：
- ✅ 第一次建议先用默认参数测试效果
- ✅ 可以同时创建多个文本节点测试不同场景
- ✅ 不满意可调整提示词重新生成

---

### Step 3：创建节点B - 面部精修

#### 3.1 连接节点A输出
- 从节点A的最佳输出结果
- 拉出新连接线向右延伸

#### 3.2 添加精修文本节点
- 在连线末端点击"+"
- 选择"文本输入"
- 命名为"`02_面部精修`"

#### 3.3 输入面部精修咒语

**方案1：使用GPT Image 2（精确控制版）**

```plaintext
把脸P好看，窄脸、缩颧骨、缩短中庭，缩短中人，缩小鼻翼，
平整光滑的皮肤，流畅的脸型，不要改变妆容，不要改变其他任何元素
```

**方案2：使用NanoBananaPro（皮肤质感版）**

```plaintext
自然美颜，精致脸型，光滑肌肤，保持原有妆容和风格不变
```

#### 3.4 选择模型

**推荐：NanoBananaPro**（优先）
- 原因：皮肤质感和细节表现最佳（⭐⭐⭐⭐⭐）
- 适用：追求极致肤质的写真作品

**备选：GPT Image 2**
- 原因：文字理解强，可精确控制每个调整项
- 适用：需要精细化定制面部特征
- 可利用"局部编辑"功能只修改面部区域

#### 3.5 参数设置

| 参数 | Banana推荐值 | Image2推荐值 |
|------|-------------|-------------|
| **模型** | NanoBananaPro | GPT Image 2 |
| **CFG Scale** | 7-8 | 7.5-8.5 |
| **Steps** | 30-40 | 35-45 |
| **Denoising Strength** | 0.3-0.4 | 0.35-0.45 |
| **蒙版范围** | 仅面部（如果有此选项）| 使用局部编辑工具选中面部 |

#### 3.6 运行与检查
- 运行生成
- 对比精修前后效果
- 检查清单：
  - [ ] 妆容是否完好保留？
  - [ ] 皮肤是否自然（非塑料感）？
  - [ ] 发型、服装、背景是否未受影响？
  - [ ] 整体风格是否与节点A一致？

---

### Step 4：创建节点C - 清晰度提升

#### 4.1 连接节点B输出
- 继续从节点B拉出连接线
- 准备最后的处理节点

#### 4.2 选择处理方式

**方式A：内置超分辨率功能（最简单）**
- 如果Shotlab提供"高清修复"或"超分辨率"按钮
- 直接应用该功能
- 设置目标分辨率：1536px 或 2048px

**方式B：Image2重生成高分辨率版本**
- 添加新文本节点
- 输入超分提示词：

```plaintext
高清修复，提升至超高清分辨率，增强细节清晰度和锐度，
降低AI痕迹，使画面接近相机原生拍摄的真实质感，
保持原有风格、色彩和构图完全不变。
```

- 模型：`GPT Image 2`
- 分辨率：选择最高可用选项
- **关键参数**：Denoising Strength设为 **0.2-0.25**（极低！）

#### 4.3 最终输出
- 运行最后一步
- 获得高清成片
- 下载保存到本地或"我的作品"

---

### Step 5：任务组与批量执行（可选但强烈推荐）

**如果Shotlab支持任务组功能**：

1. **框选所有节点**
   - 用鼠标拖选框住节点A、B、C
   - 或按住Ctrl逐个点击选中

2. **创建任务组**
   - 右键选中的节点
   - 选择"**创建任务组**"或"**新建任务组**"
   - 命名为"`3节点自动化修图流程`"

3. **配置自动执行**
   - 设置执行顺序：A → B → C
   - 确认每个节点的参数
   - 保存任务组配置

4. **一键运行与复用**
   - 点击"运行"自动执行全流程
   - 保存为模板供后续使用
   - 下次只需更换输入图片

适用场景：快速预览、样片制作、时间紧迫时

### 操作步骤

1. **进入图片生成模式**（非画布）
   - 点击顶部"**图片生成**"标签（如截图所示）

2. **上传参考图**
   - 在输入框输入 `@` 引用素材
   - 或找到上传按钮导入原图

3. **输入合并后的完整提示词**

```plaintext
电影级写实摄影质感，真实世界物理光影，低饱和高级纪实色调，
白色花瓣纷飞，暖色丁达尔发丝光，背景动态虚化，

高清修复，超分辨率，保证人物表情姿势不变，原比例。
```

4. **参数设置**
   - 模型：`GPT Image 2`
   - 分辨率：`1K 标清` 或更高
   - 其他保持默认

5. **一键生成**
   - 30秒-1分钟获得结果
   - 适合快速迭代和预览

## 五、参数优化速查表

### 5.1 不同场景的参数组合

**场景1：高质量写真交付**
```
节点A: Image2, CFG=8, Steps=50, Denoising=0.5, 分辨率=2K
节点B: BananaPro, CFG=7.5, Steps=40, Denoising=0.35
节点C: Image2, CFG=9, Steps=40, Denoising=0.22, 目标分辨率=4K
```

**场景2：快速样片预览**
```
单节点: Image2, CFG=8, Steps=35, Denoising=0.45, 分辨率=1K
```

**场景3：成本优化模式**
```
节点A: Image2, 默认参数, 分辨率=1K
节点B: Banana (非Pro版), 默认参数
节点C: 跳过（或用免费工具替代）
```

### 5.2 常见问题参数调整

| 问题现象 | 调整方向 | 具体操作 |
|---------|---------|---------|
| 人物不像原图 | 降低变化幅度 | Denoising降至0.3-0.4 |
| 变化太小/太保守 | 增加变化幅度 | Denoising提高至0.6-0.7 |
| 风格太强/不自然 | 降低风格强度 | CFG降至6-7 |
| 风格不够明显 | 增强风格强度 | CFG提高至9-10 |
| 生成太快质量差 | 增加计算步数 | Steps提高至50+
| 生成太慢 | 减少计算步数 | Steps降至25-30 |
| AI痕迹明显 | 后期处理加强 | 强化节点C，Denoising更低 |
