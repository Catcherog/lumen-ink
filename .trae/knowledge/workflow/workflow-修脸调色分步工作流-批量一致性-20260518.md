# 修脸 + 调色分步工作流（批量一致性方案）

## 核心原则

**修脸是身份稳定问题，调色是风格统一问题，必须分开处理。**

---

## 推荐流程（三步法）

### 第一步：人脸精修 / 局部修图

**目的**：保留每张照片的人物身份、五官结构、表情和真实质感，只修瑕疵、肤色、细节。

**方式**：局部重绘 / inpaint / face retouch，不要全图大幅重绘。

**提示词方向**：
```
保持人物身份、五官结构、脸型、表情、发型和光线方向不变。只进行自然人像精修：改善肤色不均、轻微淡化瑕疵、保留真实皮肤纹理，让眼神更清晰，面部光影自然。不要改变脸型、五官比例、年龄感、表情，不要过度磨皮，不要塑料皮肤。
```

**参数**：
- Denoising Strength: **0.15–0.35**（局部修五官/皮肤: 0.2–0.4）
- 不是固定Seed，每张脸细节不同可微调

**重点**：每张脸都修干净，但**不要风格化太重**。

---

### 第二步：整组统一调色

**目的**：让十几张图看起来像同一次拍摄、同一套视觉风格。

**方式**：对整图做统一风格处理。

**提示词方向**：
```
将整张图统一调整为高级人像摄影色调，低饱和、柔和对比、自然肤色、电影感光影、细腻真实质感。保持人物身份、构图、服装和背景内容不变。整体色调参考图2，像同一组写真作品。
```

**参数**：
- Denoising Strength: **0.15–0.3**
- 使用**同一套参数**
- 使用**同一张风格参考图**
- 使用**同一组提示词**

---

### 第三步：最后一轮轻微统一（可选）

**目的**：把所有成片再用很低重绘幅度过一遍，让色温、对比、颗粒、肤色更接近。

**提示词方向**（很轻）：
```
保持画面内容完全不变，只统一整体色调、对比度、肤色和光影质感。让这张图与参考图保持同一套摄影风格：柔和自然光、低饱和、高级灰、真实皮肤质感。不要改变人物五官、表情、衣服、背景和构图。
```

**参数**：
- Denoising Strength: **0.1–0.25（很低）

---

## 为什么不分步做？

| 问题 | 一步完成（修脸+调色） | 分步完成 |
|------|---------------------|---------|
| 脸部被风格化影响 | ❌ 脸可能变得不像本人 | ✅ 先锁定身份，再套风格 |
| 修饰程度不一致 | ❌ 每张图磨皮程度不同 | ✅ 修脸阶段统一标准 |
| 调色被局部干扰 | ❌ 调色结果不统一 | ✅ 调色阶段独立执行 |
| 批量失败率 | ❌ 高，返工麻烦 | ✅ 低，可控性强 |

---

## 批量处理一致性清单

处理十几张图时，调色阶段必须固定的项目：

| 项目 | 固定方式 | 说明 |
|------|---------|------|
| 风格参考图 | 同一张 | 统一色调、光影、质感 |
| 正向提示词 | 同一组 | 模板化，仅微调 |
| 负面提示词 | 同一组 | 防御体系一致 |
| 模型 | 同一个 | Image2 / BananaPro 等 |
| LoRA / 风格权重 | 固定值 | 不反复调整 |
| CFG | 固定值 | 推荐5–8 |
| 采样器 | 固定 | 同一个采样器 |
| 分辨率 | 固定 | 或长边尺寸固定 |
| Seed | 尽量固定 | 或使用相近Seed |

**注意**：修脸阶段不一定要完全固定Seed，因为每张脸细节不同。调色阶段更适合固定参数和参考图。

## 各任务流程速查

```
┌──────────────────┐
│  Step 1: 修脸    │  每张单独局部修脸，保留身份
│  Denoising 0.15–0.35  │
└─────────┬────────┘
          ↓
┌──────────────────┐
│  Step 2: 调色    │  所有图统一处理，固定参考图
│  Denoising 0.15–0.3   │
└─────────┬────────┘
          ↓
┌──────────────────┐
│  Step 3: 统一    │  低强度过一遍，检查一致性
│  Denoising 0.1–0.25   │
└──────────────────┘
```
