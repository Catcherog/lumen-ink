## 📋 文档使用说明

### 这份SOP是什么？
一份**可操作的、逐步的、标准化的**工作流程文档，指导你在Shotlab上完成个人写真的后期处理。

### 什么时候使用这份SOP？
每次处理个人写真照片时，按照此SOP执行，确保：
- ✅ 流程标准化（不会遗漏步骤）
- ✅ 质量可控（每步有检查点）
- ✅ 效率最优（避免重复和返工）
- ✅ 结果一致（不同时间处理效果稳定）

### 如何使用？
1. 打开Shotlab前，快速浏览一遍今天的任务步骤
2. 每完成一步，在对应的☐中打勾 ✓
3. 遇到问题时，查阅"问题解决速查表"
4. 处理完成后，填写"处理记录表"

## 🎯 核心原则（必须牢记）

### 原则1：质量优先于速度
>宁可多花2分钟确保质量，也不要匆忙交付有瑕疵的作品。

### 原则2：每步确认再继续
> 不要一口气跑完所有节点。每个关键节点都要停下来检查。

### 原则3：保留中间结果
> 每个节点的输出都先保存，不要覆盖。方便回退和对比。

### 原则4：记录参数和效果
> 记录你实际使用的参数和效果评价，用于持续优化。

### 原则5：遇到不确定就问AI助手
> 不要盲目尝试浪费积分。截图给我看，我会给你精准建议。

## 📝 标准处理流程（完整版）

### **阶段0：准备工作（预计2分钟）**

#### Step 0.1：环境检查 ☐
- [ ] Shotlab网站已打开并登录
- [ ] 网络连接正常
- [ ] 浏览器/客户端为最新版本
- [ ] 账户余额/积分充足（如有需要）

#### Step 0.2：素材准备 ☐
- [ ] 客户原始照片已准备好
- [ ] 照片要求检查：
  - [ ] 清晰度高（建议>1000px）
  - [ ] 面部完整且光线良好

- [ ] 照片已重命名（便于识别）：`日期_客户名_序号.jpg`

#### Step 0.3：需求明确 ☐

### **阶段2：后期修复 - 即梦模型（按需执行，预计3-15分钟）**

> ⚠️ **注意**：此阶段的详细操作依赖于你在Shotlab中探索到的即梦模型具体功能。
> 以下是通用流程，实际界面可能有所不同。

#### 🔧 决策：需要哪种类型的修复？

**诊断当前图片的主要问题**（可多选）：
- [ ] **A类问题**：有多余的元素（路人、杂物、穿帮）
- [ ] **B类问题**：有小瑕疵（面部小问题、文字错误、纹理异常）
- [ ] **C类问题**：构图不对（比例不合适、主体位置差）
- [ ] **D类问题**：需要添加新元素或重组画面
- [ ] **E类问题**：多元素拼接后风格不统一

**根据问题类型选择修复方案**：

| 问题组合 | 推荐方案 | 预计耗时 |
|---------|---------|---------|
| 只有A或B | 只用节点D（消除+修复）| 1-3分钟 |
| 有C（无D/E）| 用节点D + 节点E | 4-10分钟 |
| 有D或E | 用节点D + 节点E + 节点F | 5-15分钟 |
| 多种问题混合 | 全部节点都用 | 8-20分钟 |

---

#### 🔷 节点D：消除与简单修复（解决A/B类问题）

**适用情况**：图片中有不需要的元素或微小瑕疵

**Step D.1：使用消除画笔** ☐（如果有此功能）
- [ ] 选择"消除画笔"工具
- [ ] 用画笔涂抹要消除的区域
  - ✅ 要点：**完全覆盖**目标区域
  - ✅ 可以稍微多涂一点（宁多勿少）
  - ❌ 不要只涂一半
- [ ] 点击"消除"
- [ ] 等待3-5秒查看结果

**Step D.2：使用细节修复** ☐（强烈推荐！）
- [ ] 点击"细节修复"按钮
- [ ] 系统自动扫描并修复全图的微小问题
- [ ] 适用范围：面部崩坏、文字乱码、肢体错误、纹理异常等
- [ ] 等待几秒钟查看修复效果

**Step D.3：检查修复结果** ☐
- [ ] 消除的目标元素已消失？
- [ ] 新填充的区域自然吗？
- [ ] 细节问题已修复？
- [ ] 没有引入新问题？

```
节点D修复记录：
消除了什么：__________________________
修复了什么：__________________________
效果评价：□ 完美  □ 良好  □ 一般  □ 差
```

**✅ 节点D完成！如只有A/B类问题，可跳到【最终质检 #2】↓**

#### 🔷 节点E：重新构图（解决C/D类问题）

**适用情况**：图片比例不对、主体位置不好、想添加新元素

**Step E.1：改变图片比例（扩图）** ☐（如需要）
- [ ] 进入画布设置，将比例改为目标尺寸（如1:1→16:9）
- [ ] 图片居中显示，四周出现空白
- [ ] 使用"扩图"功能
- [ ] （可选）输入延展内容的描述词
- [ ] 运行扩图，从4个结果中选择最自然的

**Step E.2：移动主体位置** ☐（如需要）
- [ ] 回到原图，用消除画笔将主体**完全涂抹消除**
- [ ] 得到完整的干净背景图
- [ ] 复制一份原图
- [ ] 在副本上使用"一键抠图"功能
- [ ] 主体变为独立图层
- [ ] 将主体图层拖到背景图上
- [ ] 自由拖动到理想位置
- [ ] 可进行：水平翻转 / 缩放 / 旋转

**Step E.3：添加新元素** ☐（如需要）
- [ ] 导入外部素材图片
- [ ] 对素材执行一键抠图
- [ ] 将抠好的元素拖入画布
- [ ] 调整大小、位置、角度
- [ ] 安排好各元素的层次关系

```
节点E重构记录：
改变了什么：__________________________
移动了什么：__________________________
添加了什么：__________________________
新构图满意度：□ 很满意  □ 满意  □ 一般  □ 不满意
```

**✅ 节点E完成！↓**

#### 🔷 节点F：风格统一（解决E类问题）

**适用情况**：多元素拼接后边缘僵硬、光影不一致、风格违和

**方案选择**：

**方案F-A：熔图（简单快速）** ☐
- [ ] 确认只是简单的边缘过渡问题
- [ ] 使用"熔图"功能
- [ ] 调整参数：强度、边缘羽化等
- [ ] 执行熔图
- [ ] 检查融合效果

**方案F-B：图生图+图片信息参考模式（高质量推荐）** ☐ ⭐
- [ ] 确保所有元素已摆好位置（得到草图）
- [ ] 使用"图生图"功能
- [ ] 选择"**图片信息参考模式**"
- [ ] 设置**参考强度**：`____`
  - 0.8-0.9：完全保留元素位置
  - **0.5-0.7：平衡保留与风格（最常用）** ⭐
  - 0.2-0.4：大改风格但保留构图
- [ ] （可选）输入风格提示词：
  ```
  电影质感，光影自然，色彩和谐，
  画面统一，边缘过渡自然，高质量渲染
  ```
- [ ] 运行生成
- [ ] 从4个结果中选择最佳的
- [ ] （可选）最后再用一次细节修复收尾


## 🆘 问题解决速查表

### 快速定位问题

**问题现象 → 可能原因 → 解决方案**

| 问题现象 | 最可能的原因 | 快速解决方案 |
|---------|------------|-------------|
| 人物不像本人 | Denoising过高 | 降低至0.4以下 |
| 皮肤塑料感 | 模型选择不当 | 改用BananaPro |
| 场景元素突兀 | 描述词不够柔和 | 加"少量"、"若隐若现" |
| AI痕迹明显 | 缺少超分或超分不够 | 强化节点C，Denoising更低 |
| 生成时间过长 | Steps过多或分辨率太高 | 降低Steps至30-40 |
| 消除后有残留 | 涂抹不完全 | 重新涂抹，扩大范围 |
| 抠图边缘不准 | 原图对比度不够 | 先调对比度再抠图 |
| 熔图后仍不自然 | 问题超出熔图能力 | 改用图生图+参考模式 |
| 风格变化太大 | 参考强度过低 | 提高至0.7-0.9 |
| 风格还是不统一 | 参考强度过高 | 降低至0.5-0.6 |

### 何时该寻求帮助？

**立即召唤AI助手的情况**：
- ❌ 连续2次尝试同一问题仍未解决
- ❌ 出现从未见过的新错误或异常
- ❌ 不确定应该选择哪个参数或方案
- ❌ 对效果不满意但不知道如何改进
- ❌ 想要尝试新的创意或技巧但不知道怎么做

**召唤方式**：
- 直接告诉我当前的状态和问题
- 附上截图（如果涉及界面操作）
- 描述你已经尝试过的方法
- 我会给出精准的下一步指导

## 📊 质量评分标准参考

### 评分维度说明（供自评参考）

**人物特征保留（权重30%）**：
- 10分：完全像本人，亲友一眼能认出
- 8-9分：非常像，细微差别可接受
- 6-7分：比较像，但某些特征有变化
- 4-5分：不太像，需要仔细辨认
- 1-3分：完全不像，无法辨认是同一人

**场景融合自然度（权重25%）**：
- 10分：完美融合，看不出是合成的
- 8-9分：非常自然，专业人士才能看出
- 6-7分：基本自然，仔细看有些许痕迹
- 4-5分：较明显的人工合成痕迹
- 1-3分：明显的拼贴感，很不自然

**面部效果质量（权重25%）**：
- 10分：完美肤质，精致五官，妆容完好
- 8-9分：很好，有小瑕疵但不影响
- 6-7分：一般，有明显可改进的地方
- 4-5分：较差，皮肤假或五官变形
- 1-3分：很差，严重失真或崩坏

**整体画质与技术指标（权重20%）**：
- 10分：超高清晰度，无AI痕迹，印刷级品质
- 8-9分：高清，AI感极淡，商业展示可用
- 6-7分：中等清晰，有一定AI感，网络分享够用
- 4-5分：较低清晰，AI感明显，仅预览用
- 1-3分：模糊或有严重技术缺陷
