## 四、工作方式偏好

### 4.1 自动化程度：**半自动模式** ⭐核心偏好

**明确表述**："我给图片，AI处理，我确认效果"

**工作流程特点**：

```
用户操作                    AI/系统处理              用户确认
───────                    ──────────              ────────
上传图片/输入需求   →      AI自动生成/修复    →      用户审核结果
                                                    ├─ 满意 → 确认/保存
                                                    └─ 不满意 → 调整/重做
```

**关键特征**：
- ✅ 用户负责：提供素材、提出需求、确认最终效果
- ✅ AI负责：执行生成/修复任务
- ✅ **每一步都需要人工确认**（不是全自动批量）
- ✅ 强调**可控性**和**质量保证**
- ✅ 不追求无人值守的全自动化

### 4.2 整合策略：**以Shotlab为主**

**已确认的选择**：方案A - Shotlab为主平台

**原因**（推测）：
- Shotlab功能完整（画布+多模型+参考生图）
- Shotlab有即梦的模型（无需切换平台）
- 已经熟悉或正在学习Shotlab的操作
- 希望在一个平台上完成所有工作

- **类型**：韩系高级奶油肌（非哑光、非陶瓷）
- **要求**：肤色提亮半到一档、均匀肤色、去除暗沉瑕疵
- **底线**：必须保留真实皮肤纹理和毛孔、保留面部立体光影
- **禁止**：塑料皮、过度磨皮、假白、柔焦糊脸

### 11.3 修脸原则

- **程度**：轻微调整、自然为主
- **方向**：小V脸、下颌线收紧、中庭缩短、鼻翼鼻头缩小、人中缩短
- **底线**：保持本人特征、不要网红脸、不要过度整形感、五官辨识度不变

### 11.4 提示词方法论

- **结构**：六层模型（定位→身份→脸部→皮肤→调色→输出）
- **策略**：攻守兼备（正向+负面双重锁定）
- **关键**：反复强调核心约束、逐项量化调整程度
- **防御**：17项负面提示词覆盖所有翻车点

### 11.5 四大翻车点认知

| 翻车点 | 防御关键词 |
|--------|-----------|
| 脸修成网红 | 轻微、保持本人特征、不要网红蛇精脸 |
| 肤色太白 | 保留真实纹理、不要假白、不要塑料皮 |
| 背景被重绘 | 保留背景构图、不要改变背景结构 |
| 光影变滤镜 | 保留面部立体光影、不要柔焦糊脸 |

### 11.6 提示词输出方式（重要约束）

**核心原则：方法论内化，按需定制，拒绝照抄模板**

- ❌ **禁止**：把知识库中的提示词原文照搬输出
- ✅ **要求**：理解方法论后，根据每张照片的具体情况灵活生成
- ✅ **流程**：分析照片特点 → 听取用户具体需求 → 融合方法论 → 输出定制化提示词
- ✅ **变化**：结构、措辞、强调点、负面词都应根据照片和需求调整
- ✅ **减法**：不需要的部分就删掉，不要为了"完整"而堆砌

## 十二、重要方法论积累（2026-05-18新增）

### 12.1 img2img 修图核心认知

**核心理念**：
> img2img 让模型"看着原图改"，而不是从零画一张图。

**四大锁定原则**：
| 要素 | 作用 |
|------|------|
| 原图 | 锁身份和结构（人物、姿势、构图、光线） |
| 遮罩 | 锁修改范围（只改遮罩区域） |
| 参考图 | 锁风格（色调、光影、质感） |
| 提示词 | 锁边界（说清楚不能改什么） |

### 12.2 分步修图方法论

用户明确确认的修图策略：
1. **修脸 → 调色 → 统一**（三步分步做，绝不一步完成）
2. **修脸是身份稳定问题，调色是风格统一问题，分开处理**
3. 局部重绘（Inpaint）比全图 img2img 更稳定
4. 批量一致的关键：调色阶段固定所有参数和参考图

### 12.3 提示词分层写法

用户确认的标准写法：
- **保留**：保持原图哪些不变
- **修改**：只改什么区域、改成什么
- **风格**：参考什么风格/参考图
- **限制**：不能改什么

### 12.4 参考图分类使用

| 类型 | 用途 |
|------|------|
| 人物原图 | 锁定身份、脸、姿势、构图 |
| 风格参考图 | 色调、光影、质感、摄影风格 |
| 服装参考图 | 只参考衣服款式 |
| 背景参考图 | 只参考环境氛围 |
